OSI and TCP/IP Models
  • The diagram illustrates the OSI Model, the TCP/IP Model, and how the 2 models correspond.
  • The OSI Model:
    • Application
    • Presentation
    • Session
    • Transport
    • Network
    • Data Link
    • Physical
  • The TCP/IP Model:
    • Application
    • Transport
    • Internet
    • Network Access
  • The 2 models correspond as follows:
    • Application, Presentation, and Session (OSI) correspond to Application (TCP/IP).
    • Transport (OSI) corresponds to Transport (TCP/IP).
    • Network (OSI) corresponds to Internet (TCIP/IP).
    • Data Link and Physical (OSI) correspond to Network Access (TCP/IP).
